# Anna Gen 2 — VRM Viewer

Quick scaffold for a minimal VRM viewer using three and @pixiv/three-vrm.

Run:

```bash
cd anna-web/anna-gen2
npm install
npm run dev
```

Open the URL shown by Vite (usually http://localhost:5173).

Notes:
- Place a VRM file at `models/sample.vrm` or paste a remote VRM URL into the input and click "Load VRM".
- This project uses Vite so ESM imports from `three` and `three/examples` work.
